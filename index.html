수치 미분과 정보과학

순간 변화율을 이용해 미분을 계산하는 방법과
이를 컴퓨터가 어떻게 근사하여 활용하는지 소개

1. 수치 미분이란 무엇인가?

미분은 함수의 순간 변화율, 즉 특정 순간의 기울기를 의미한다.
미분의 정의는 다음과 같다.

f'(x)=lim₍h→0₎ [f(x+h)-f(x)] / h

하지만 이 정의는 극한 개념을 포함하므로 컴퓨터는 그대로 계산할 수 없다.
그래서 실제 계산에서는 작은 h를 써서 기울기를 근사하며,
이를 수치 미분(Numerical Derivative)이라고 한다.

2. 차분 근사

차분 근사는 수치 미분의 기본 원리이다.
대표적으로 전진 차분, 후진 차분, 중심 차분이 있으며
각 수치 미분 근사식은 다음과 같다:

전진 차분: f'(x) ≈ [f(x+h)-f(x)] / h

후진 차분: f'(x) ≈ [f(x)-f(x-h)] / h

중심 차분: f'(x) ≈ [f(x+h)-f(x-h)] / (2h)

*이 때, 중심 차분은 양쪽 값을 모두 사용하기 때문에 보통 가장 정확하다.

3. 컴퓨터에서 계산 시

이론적으로는 h가 작아질수록 도함수에 가까워지지만,
컴퓨터는 숫자를 근사된 형태로 저장하므로 문제가 발생한다.

  • h가 너무 크면 근사가 부정확해진다.
  • h가 너무 작으면 부동소수점 오차로 인해 부정확해진다.

→ 컴퓨터에는 적당한 h가 필요하다.

4. 컴퓨터가 극한을 계산할 수 없는 이유

컴퓨터는 실수를 2진수로 근사해 저장하기 때문에 무한히 작은 값을 표현할 수 없다.
따라서 극한을 직접 계산할 수 없으므로 차분을 이용한다.

5. 머신러닝에서의 수치 미분

순간 변화율은 인공지능이 학습할 때도 사용된다. AI는 문제를 잘 풀기 위해 손실 함수라는 값을 줄여야 하는데,
이때 손실 함수의 순간 변화율이 값의 변화 방향을 결정한다.

  • 기울기가 양수 → 손실이 증가하는 방향 → 반대쪽으로 이동
  • 기울기가 음수 → 손실이 감소하는 방향 → 그쪽으로 이동

AI는 이 과정을 반복하면서 점점 더 정확해진다.
즉, AI의 학습 과정은 순간 변화율을 이용해 정확해지는 과정이라고 생각할 수 있다.

6. 실제 데이터 분석에서의 수치 미분

대부분의 데이터(기온·속도·주가·심박수 등)는 일정한 시간 간격으로 측정된 이산 데이터이다.
→ 연속 함수가 아니기 때문에 직접 미분을 할 수 없다.
그러므로 다음과 같이 수치 미분을 사용한다:

f'(t) ≈ [f(t + Δt) - f(t)] / Δt

이 근사 변화율이 사용되는 예시로 몇 가지를 들어보면,

  • 주가 급등·급락 시점 파악
  • 심박수 급증 감지
  • 자율주행차의 속도·가속도 계산
  • 기온 변화 속도 분석

등이 있다.

수치 미분 방식 비교

위 그래프에서 각 직선은 다음과 같다:
검은색 곡선: 주어진 함수
파란색(점 A를 지나는 직선): 전진 차분
보라색(점 B를 지나는 직선): 중심차분
빨간색(점 C를 지나는 직선): 접선
주황색(점 D를 지나는 직선): 후진차분
